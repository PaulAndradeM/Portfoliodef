Resumen

En esta aplicación se permite llevar un registro completo de las ventas, pudiendo visualizar los productos vendidos, sus precios, cantidades y totales de ventas. Un proyecto desarrollado en un tiempo de 5 semanas y se completó con una gran cantidad de funcionalidades. Entre ellas el inicio de sesión independente de cada tienda subido en la base de datos, la posibilidad de cambiar el precio de cada producto y agregar mas según cada tienda, la posibilidad de ver los productos de esa tienda y su código y la actualización diaria de los precios de los productos según el mercado.

Este es un proyecto de desarrollo de aplicacion desktop que se ha realizado en Python y MongoDB. siendo un proyecto de la StartUp Rubiant, de la que hago parte. En este proyecto desempeñe el cargo de desarrollador pirncipal y creando interfaces de usuario para la aplicación y el servidor, Por ello Fue un proyecto FullStack, incluyendo todo el FrontEnd y BackEnd.
