Mis Habilidades

Pon el cursor sobre la tecnología para ver su resumen.

HTML: Lenguaje de marcado para estructurar el contenido de las páginas web, usando etiquetas como párrafos, enlaces e imágenes.
CSS: Lenguaje de hojas de estilo para definir la apariencia de las páginas web, incluyendo colores, fuentes y diseño.
JavaScript: Lenguaje de programación que se utiliza para crear interactividad y funcionalidades dinámicas en las páginas web. Permite manipular el DOM, manejar eventos y realizar solicitudes a servidores, entre otras tareas.
React: Biblioteca de JavaScript para construir interfaces de usuario mediante componentes reutilizables y gestión eficiente del estado.
SASS: Extensión de CSS que añade funcionalidades avanzadas como variables y mixins para un diseño más eficiente.
Git: Sistema de control de versiones para gestionar y rastrear cambios en el código fuente, facilitando la colaboración entre desarrolladores.
TypeScript: Superconjunto tipado de JavaScript que añade tipado estático y otras características avanzadas. Mejora la robustez y mantenibilidad del código, permitiendo detectar errores en tiempo de desarrollo.
Python: Lenguaje de programación interpretado, de alto nivel y multiparadigma. Destaca por su sintaxis clara y legible, amplia biblioteca estándar y versatilidad en diferentes áreas como desarrollo web, análisis de datos e inteligencia artificial.
